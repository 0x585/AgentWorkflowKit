## 变更说明

- `post-commit` 不再执行裸 `git push`，而是直接调 `session_push_autorelease.sh`，把首次 push、merge、push main 和 downstream apply 收口到单一入口。
- `session_push_autorelease.sh` 与 `session_release_resume.sh` 增加了更明确的 conflict / missing state / pending merge 提示，输出 source branch、target branch、primary repo、source worktree 等恢复上下文。
- 同步更新中央维护文档、模板工件和 release `1.0.27`，并新增测试覆盖新入口和恢复提示。
